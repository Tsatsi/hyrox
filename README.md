# HYROX Workout Planner

A React app for HYROX race preparation — tracks your training week, meal plan, race countdown, and crew progress.

**Target event:** HYROX Johannesburg · Nasrec Expo Centre · Nov 28–29, 2026

## Features

- **Countdown** — live countdown to race day with your name personalisation
- **Workout Week** — weekly training plan with session tracking
- **Meal Week** — structured meal plan for race prep nutrition
- **My Crew** — share a group link so your crew can track each other's progress

## Stack

- React 18 + Vite
- Tailwind CSS
- Vitest for unit tests
- localStorage for persistence (no backend required)

## Getting Started

```bash
npm install
npm run dev
```

## Commands

| Command | Description |
|---|---|
| `npm run dev` | Start dev server |
| `npm run build` | Production build |
| `npm run preview` | Preview production build |
| `npm test` | Run tests once |
| `npm run test:watch` | Run tests in watch mode |

## Altitude Notes

Johannesburg sits at ~1,750m. Start 10–15 sec/km slower than sea-level pace and hydrate 3.5–4.5 L/day.
